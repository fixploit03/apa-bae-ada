# Setup Adapter Wi-Fi

## Langkah-Langkah

<h4>1. Buka VirtualBox.</h4>
<h4>2. Pilih mesin yang mau di colokin adapter Wi-Fi (Kali Linux).</h4>
<img src="https://github.com/fixploit03/apa-bae-ada/blob/main/wireless%20hacking/preparation/setup%20adapter%20wifi/img/1.png" />
<h4>3. Klik menu Settings.</h4>
<img src="https://github.com/fixploit03/apa-bae-ada/blob/main/wireless%20hacking/preparation/setup%20adapter%20wifi/img/2.png" />
<h4>4. Klik tab USB, terus seting kaya gini.</h4>
<img src="https://github.com/fixploit03/apa-bae-ada/blob/main/wireless%20hacking/preparation/setup%20adapter%20wifi/img/3.png" />
<h4>5. Klik icon USB kecil yang ada tanda ples ijo, terus pilih adapter yang mau di colok.</h4>
<img src="https://github.com/fixploit03/apa-bae-ada/blob/main/wireless%20hacking/preparation/setup%20adapter%20wifi/img/4.png" />
<h4>6. Jalanin mesin.</h4>
<h4>7. Buka terminal, terus ketikin <code>lsusb</code> buat ngeliat semua USB yang kecolok di mesin.</h4>
<img src="https://github.com/fixploit03/apa-bae-ada/blob/main/wireless%20hacking/preparation/setup%20adapter%20wifi/img/5.png" />
<h4>8. Ketikin <code>iwconfig</code> buat ngeliat semua interface yang ada di mesin.</h4>
<img src="https://github.com/fixploit03/apa-bae-ada/blob/main/wireless%20hacking/preparation/setup%20adapter%20wifi/img/6.png" />
