# Setup Adapter Wi-Fi

## Langkah-Langkah

<div align="center">
  <h4 align="left">1. Buka aplikasi VirtualBox.</h4>
  <h4 align="left">2. Klik OS Kali Linux, terus klik Settings.</h4>
  <img src="https://github.com/fixploit03/apa-bae-ada/blob/main/wireless%20hacking/setup%20adapter/img/1.png" />
  <h4 align="left">3. Klik USB.</h4>
  <img src="https://github.com/fixploit03/apa-bae-ada/blob/main/wireless%20hacking/setup%20adapter/img/2.png" />
  <h4 align="left">4. Klik icon USB kecil yang ada tanda ples ijo ijonya, terus lu pilih adapter yang mau lu pake.</h4>
  <img src="https://github.com/fixploit03/apa-bae-ada/blob/main/wireless%20hacking/setup%20adapter/img/3.png" />
  <h4 align="left">5. Klik OK.</h4>
  <img src="https://github.com/fixploit03/apa-bae-ada/blob/main/wireless%20hacking/setup%20adapter/img/4.png" />
  <h4 align="left">6. Jalanin Kali Linux nya, terus buka terminal.</h4>
  <h4 align="left">7. Ketikin perintah <code>lsusb</code> buat ngeliat list usb yang kecolok di mesin lu.</h4>
  <img src="https://github.com/fixploit03/apa-bae-ada/blob/main/wireless%20hacking/setup%20adapter/img/5.png" />
  <h4 align="left">8. Ketikin perintah <code>iwconfig</code> buat ngecek apakah interface wlan lu ada apa kaga.</h4>
  <img src="https://github.com/fixploit03/apa-bae-ada/blob/main/wireless%20hacking/setup%20adapter/img/6.png" />
  <h4 align="left">9. Dari output gambar di atas, lu punya 1 interface wlan. Yaitu <code>wlan0</code>.</h4>
</div>
