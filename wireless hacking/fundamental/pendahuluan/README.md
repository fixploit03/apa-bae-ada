# Pendahuluan

Sebelum masuk ke wireless hacking, lu harus kenal dulu sama yang namanya Wi‑Fi, mulai dari apa itu Wi‑Fi, gimana cara kerjanya, apa aja komponennya, sampai istilah‑istilah dasarnya, tujuannya biar pas lu masuk ke dunia wireless hacking lu kaga bengong atau planga‑plongo, tapi bener‑bener paham apa yang lagi lu kerjain.
